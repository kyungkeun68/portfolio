고객이 직면한 문제

  • 학생이 수학과목을 다른 과목에 피해 어렵게 느끼는 이유로 "수학공부를 하지 않기 때문"이라는 응답(28.1%)이 가장 많고 그 다음으로는 "개념이나 성질들 이해하기 힘든 내용이 있기 때문"이라는 응답(23.6%)이 많음

시장 상황

  • 중고등학교 수학은 정확한 개념 이해와 유형별 문제 풀이를 통해 성적 향상을 이뤄낼 수 있다는 인식이 자리 잡음
  • 각종 문제집과 학원에서는 그들만의 개념과 유형 학습을 개발하여 학습을 제공
  • 그러나 대부분의 B2C 수학 학습 서비스는 콘텐츠의 퀄리티보다는 다양한 방식의 동기부여에 더 집중하고 있음
💡

B2C 수학 학습 서비스는 게임요소, 포인트, 리워드 등을 주요 차별포인트로 내세우고 있음

수학대왕

  • 수학 자기주도학습을 위한 문제, 강의, 학습관리 제공
    • 개념 학습과 유형별 문제 풀이가 독립적으로 존재
    • 유형별 문제풀이에 치우친 학습 방식
  • 차별포인트
    • 랭킹제도, 중/고등학교 구분하여 운영 (학습랭킹/실력랭킹/학교랭킹)
    • 환급제도 : 하루 10문제 풀면 최대 1,000원 환급, 1년 최대 37만원 환급가능
    • 내신 및 모의고사 기출 제공
    • 나와의약속 : 시간과 요일을 정하면 정해진 타이밍에 공부 알림전송
    • 스터디윗미 : 현재 앱에서 공부중인 회원 리스트, 회원별 공부시간 표기

디북피플

  • 천재교육 문제집을 통해 수학공부하는 앱 (교재의 틀에서 벗어나지 못하는 콘텐츠)
  • 차별포인트
    • 문제풀이 화면에서 난이도, 정답률, 풀어본 사람, 이전 결과 정보제공
    • 챌린지 설정 : 선택한 교재에 대하여 개월 단위 기간 설정
    • 커뮤니티 : 교재 별 댓글게시판 운영 (게시글 100개 이상 게시판 4개, 최다 게시판은 9000개 이상)
    • 무료 (추후 유료 전환 예정)
    • 문제를 풀거나 챌린지 완료할때 마다 포인트 지급, 포인트를 통해 상품 응모 가능

매쓰피드

  • 웅진싱크빅에서 만든 AI 기반 초중등 연산 앱 (교육과정을 따라가는 것이 아닌 연산 연습 위주)
  • 차별포인트
    • 수학 카메라
    • AI ON/OFF 선택 가능, AI ON은 유료 (학습내용 바탕으로 수준진단, 취약점, 리포트 제공)
    • 프린트 가능한 워크시트 제공
    • 글로벌 서비스로서 풀어본 문제가 동일해도 선택 국가에 따라 진도율 다르게 표기
    • 뱃지 퀘스트 : 앱 내 활동으로 구성
    • 활동 내역에 따라 '날개' 획득, 날개를 이용하여 캐릭터나 아이템 구매(오픈 예정)

🧭 기획 목적 및 목표

기획 목적

  • 혼자서도 수학 공부를 할 수 있는 콘텐츠/커리큘럼을 만든다.
  • 효율적인 개념/유형 학습을 할 수 있는 콘텐츠/커리큘럼을 만든다.

기획 목표

  • 중1부터 고3까지 전 과목의 커리큘럼을 구성할 수 있는 좋은 콘텐츠 생산
    • 개념 카드, 개념 OX 각각 1000여 개, 유형 카드 2000여 개 중 고등 수학2와 선택 과목 3종 담당
    • 콘텐츠 오류 신고 비율 0.1% 미만으로 유지
  • 고객이 스스로 학습을 이어나갈 수 있도록 유도하는 학습 및 분석 로직 구성

🏗 콘텐츠 제작 과정

콘텐츠 구성 요소

  1. 커리큘럼 학습의 기반이 되는 콘텐츠 제작
    1. Easy모드 (개념 위주의 학습)
      • 개념카드 : 소단원 별 문제를 풀기 위해 필요한 개념을 정리한 콘텐츠, 주요 용어에 빈 칸을 만들어 학습 후 해당 개념을 잘 이해하고 있는지 확인할 수 있게 구성
      • 개념OX : 헷갈리는 개념을 올바르게 설명할 수 있는지 확인하는 OX문제 형식의 콘텐츠, 등하교나 짧은 시간에 모바일 기기로 빠르게 풀어볼 수 있게 구성
    2. Normal모드 (유형 위주의 학습)
      • 유형카드 : 유형에 속해 있는 문제들을 효과적으로 풀 수 있도록 설명해주는 콘텐츠
  2. 학습 및 분석 로직 구성
    1. 드릴 학습 (특정 컨셉과 단원의 문제를 무제한으로 풀 수 있는 학습 서비스)
      • 눈으로 풀기 : 개념과 관련된 문제를 풀 수 있도록 로직 구성 (개념 빈칸 채우기, OX퀴즈)
      • 기본기 다지기 : 가장 쉬운 난이도의 문제를 풀 수 있도록 로직 구성
      • 실전 연습하기 : 시험 대비에 적합한 문제가 출제되도록 로직 구성
      • 고득점 달성하기 : 최상위권 학생들에게 추천하는 고난도 문제가 출제되도록 로직 구성
    2. 분석 로직 구성 및 화면 기획
      • 성적향상을 이루기 위해서는 적절한 공부 방법 및 방향성이 필요하므로, 분석 페이지가 다음 학습을 연결해주는 역할을 하도록 기획
      • 전체 유저 중 해당 유저의 성적(상위 nn%)을 보여주고, 더 높은 위치에 올라갈 수 있는 방법을 제시하는 형태로 구성

🚀 결과물 및 목표 달성 여부

결과물

  • 수학2와 고등 수학 선택 3종(확률과 통계, 미적분, 기하)의 개념카드, 개념OX 콘텐츠 총 862개 제작
  • 초기 분석 페이지 화면 기획 및 적용하였으나, 데이터 문제로 분석 뒤 이어지는 학습이 관련 문제 추천으로 제한 → 최초 기획 의도가 충분히 달성되지 않았다고 판단하여 추후 분석 페이지를 재검토하여 새로운 기능을 기획하고자 함

목표 달성 여부

오픈 120일 기준

  • 전체 고객의 일 풀이 수 최대 15,400건, 총 누적 풀이 수 110만건 이상 달성
  • 문제 오류 신고 수 500건으로 풀이수 대비 오류 신고 비율 0.04%를 기록
